# 唐宋诗仙 - 李白与苏轼

一个展示唐代诗仙李白与北宋文豪苏轼生平与作品的静态网站。

## 📖 项目简介

本网站旨在讲述中国文学史上最璀璨的双星：
- **李白**（701-762）- 盛唐浪漫主义诗歌巅峰，被誉为"诗仙"
- **苏轼**（1037-1101）- 北宋文学最高成就代表，被称为"坡仙"

二人相隔三百余年，却同样以豪放不羁的个性和卓越超群的才华，流传千古。

## 📁 项目结构

```
life-sushi/
├── index.html              # 首页 - 双诗人介绍
├── libai-index.html        # 李白主页
├── libai-timeline.html     # 李白生平时间线
├── libai-works.html        # 李白代表作品
├── dongpo-index.html       # 苏轼主页
├── dongpo-timeline.html    # 苏轼生平时间线
├── dongpo-works.html       # 苏轼代表作品
├── dongpo-evaluation.html  # 苏轼历史评价
├── comparison.html         # 李白与苏轼对比
├── css/
│   └── style.css           # 样式文件（双主题色）
└── README.md               # 项目说明
```

## 🎨 设计特点

- **双主题配色**：李白（蓝色系）+ 苏轼（棕色系）
- **对比展示**：二人时代、成就、性格的多维度对比
- **时间线叙事**：清晰呈现各自人生重要节点
- **作品赏析**：精选诗词代表作，附创作背景
- **响应式设计**：适配桌面和移动设备
