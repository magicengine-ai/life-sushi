# 苏轼生平网站 - Life of Su Shi

一个展示北宋文学家苏轼（苏东坡）生平的静态网站。

## 📖 项目简介

本网站旨在讲述苏轼（1037-1101）的传奇一生，包括：
- 早年经历与进京应试
- 宦海沉浮与政治生涯
- 乌台诗案与被贬黄州
- 文学成就与代表作品
- 历史评价与文化遗产

## 🌐 在线预览

网站将部署在 GitHub Pages：
https://magicengine-ai.github.io/life-sushi/

## 📁 项目结构

```
life-sushi/
├── index.html          # 首页 - 苏轼简介
├── timeline.html       # 生平时间线
├── works.html          # 代表作品
├── evaluation.html     # 历史评价
├── css/
│   └── style.css       # 样式文件
└── README.md           # 项目说明
```

## 🚀 部署到 GitHub Pages

```bash
# 初始化 git 仓库
git init
git add .
git commit -m "Initial commit: 苏轼生平网站"

# 添加远程仓库
git remote add origin git@github.com:magicengine-ai/life-sushi.git

# 推送到 GitHub
git push -u origin main
```

## 🎨 设计特点

- **中国风配色**：采用棕色、橙色系，呼应宋代美学
- **响应式设计**：适配桌面和移动设备
- **时间线展示**：清晰呈现苏轼人生重要节点
- **作品赏析**：精选诗词散文代表作
